# Z915

An nRF52840 (MDBT50Q) wireless mechanical keyboard that fits in the chassis and recreates the functions of the Logitech G915. ZMK is used as the firmware.
G915 features to be recreated
  - Per-key RGB LED
  - Volume scroll wheel
  - Wireless (BLE)
  - Battery level reporting

Features to be added/upgraded
  - USB-C charging (G915 has micro-B USB)
  - Upgrade the feel of the rubber dome switches by using snap dome switches underneath the rubber
  - ZMK-included features
